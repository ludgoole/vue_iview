vue-iview脚手架
=============


## 项目运行说明
- 克隆项目
```
git clone git@github.com:ludgoole/vue_iview.git
```
- 安装依赖包
```
cnpm install / npm install
```
- 启动项目
```
cnpm run dev
```
- 项目打包
```
cnpm run build
```

## 框架
- [UI框架 iview](https://www.iviewui.com)
- [JS框架 vue](https://cn.vuejs.org)

## 项目目录说明
```
build       --- 脚手架相关配置目录
config      --- 开发和构建两种环境下的配置
src         --- 项目核心代码文件夹
|api        --- 接口相关配置
|assets     --- 静态资源将会被打包
|components --- 公共组件
|view      --- 相关页面目录
|router     --- 路由配置相关
|store      --- 状态管理部分
|utils      --- 公共方法类
static      --- 静态资源将不会被打包
test        --- 测试相关配置
```
